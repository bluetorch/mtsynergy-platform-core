## State Definitions

### PROJECT_PHASE

- `UNINITIATED`: Framework installed, project not started
- `INITIALIZING`: START phase in progress
- `DEVELOPMENT`: Active development using RIPER workflow
- `MAINTENANCE`: Long-term maintenance mode

### RIPER_CURRENT_MODE

- `NONE`: No active mode (only valid when UNINITIATED/INITIALIZING)
- `RESEARCH`: Information gathering
- `INNOVATE`: Brainstorming approaches
- `PLAN`: Creating specifications
- `EXECUTE`: Implementing changes
- `REVIEW`: Validating implementation

### START_PHASE_STATUS

- `NOT_STARTED`: START phase has not begun
- `IN_PROGRESS`: Currently in START phase
- `COMPLETED`: START phase finished

### START_PHASE_STEP

- `0`: Not started
- `1`: Requirements Gathering
- `2`: Technology Selection
- `3`: Architecture Definition
- `4`: Project Scaffolding
- `5`: Environment Setup
- `6`: Memory Bank Finalization
